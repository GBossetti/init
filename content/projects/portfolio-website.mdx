# Portfolio Website

This is the website you're currently viewing. It serves as my personal brand and showcases my work as a software developer.

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS v4
- **Content**: MDX with Velite
- **Hosting**: Vercel

## Features

- Dark theme with techy aesthetic
- Type-safe content management
- Syntax-highlighted code blocks
- SEO optimized
- Fully responsive

## What I Learned

Building this project helped me understand:

1. Modern Next.js patterns with the App Router
2. Type-safe content management with Velite
3. Tailwind CSS v4 features
4. MDX for rich content authoring
